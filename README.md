## 🎯 Tujuan Repository

Repository ini dibuat untuk:

- Menyimpan materi perkuliahan dan praktikum.

# 📖 Mata Kuliah Praktikum

## 🌐 Jaringan Komputer (Jarkom)

Mata kuliah yang mempelajari konsep dasar hingga implementasi jaringan komputer.

### Materi yang Dipelajari

- Model OSI dan TCP/IP
- Pengalamatan IP (IPv4 & IPv6)
- Subnetting dan VLSM
- Routing dan Switching
- VLAN
- DHCP dan DNS
- Network Security
- Wireless Network
- Monitoring dan Troubleshooting Jaringan

### Praktikum dan Laboratorium

- Konfigurasi IP Address
- Simulasi Jaringan menggunakan Cisco Packet Tracer
- Konfigurasi Routing Static dan Dynamic
- Implementasi VLAN
- Konfigurasi DHCP Server
- Konfigurasi DNS Server
- Analisis Trafik Jaringan
